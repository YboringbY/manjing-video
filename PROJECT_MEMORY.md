# 漫镜视频项目记忆与进度文档

更新时间：2026-06-14

## 1. 项目定位

项目名称：**漫镜视频**

当前定位：面向短剧团队的 AI 视频生产工作台，用于完成项目创建、剧本导入、提示词拆分、分镜生成、丽帧资产调用、Seedance 视频生成、生成任务管理、视频资产管理。

当前仍是本地 MVP，但已经具备真实 Seedance 2.0 视频生成和丽帧资产库查询/选择能力。

## 7. 分镜拆分逻辑

### 7.1 背景问题

用户输入一整段 12s 视频提示词后，生成出来的视频像是把完整 12s 剧情压缩进 3-4s，动作和台词太快。

原因：旧分镜点击“重生成”时，仍然直接拿旧的完整 prompt 去生成。

### 7.2 当前修复

当前已增加：

```ts
splitLongPromptIntoShots()
splitExistingShot()
generateShotOrSplit()
```

新建分镜时，如果输入长 prompt 且时长 >= 10 秒，会自动拆成 4 段。

旧分镜点击 `生成 / 重生成` 时，如果检测到还是完整长 prompt，会先自动拆成 4 条分镜，再分别生成。

分镜表中还有手动按钮：

```text
拆4段生成
```

### 7.3 分段 prompt 格式

拆分后每段会带类似约束：

```text
这是一个 12 秒完整镜头的第 1/4 段，只生成本段内容，不要重复完整镜头。
本段目标时长：4秒。
当前段落动作和台词：...
节奏要求：对白和动作自然，不要压缩全部剧情，不要把其他段落内容提前生成。
只表现当前段落，不要出现其他段落的动作或台词。
真人写实短剧质感，电影级布光，禁止生成字幕。
```

目标：

- 不再把完整 12s 剧情压进一个短视频。
- 每条分镜只生成自己负责的段落。
- 让动作和台词节奏更自然。

### 7.4 当前限制

当前拆分仍是规则拆分，不是 AI 语义拆分。

主要依赖：

```text
时间轴
标点
然后 / 接着 / 随后 / 同时
```

最推荐用户输入格式：

```text
0-3秒：主管停在林凡面前，眼神嫌弃。
3-6秒：主管逼近训斥：“不干活在这发什么呆呢？”
6-9秒：林凡低头沉默，握紧拳头。
9-12秒：林凡抬头反击，气氛紧张。
```

后续建议接 AI 分镜拆分器。

```text

```

注意：这不是严格换脸模型，人物一致性仍取决于 Seedance 对参考图的遵循程度。

## 9. Seedance 视频生成

### 9.1 创建任务

接口：

```text
POST /api/video-tasks
```

后端调用：

```text
POST /ai-open-platform-api/v1/lz/video/task/create
```

核心 payload：

```ts
{
  prompt,
  mode: "fast",
  resolution: "720p",
  ratio,
  duration,
  generate_audio: true,
  watermark: false,
  input_type,
  generation_type: "video",
  images,
  videos,
  audios
}
```

素材数量限制：

```text
图片最多 9 个
视频最多 3 个
音频最多 3 个
```

时长限制：

```ts
if duration < 4 return 4
if duration > 15 return 15
```

### 9.2 状态查询

接口：

```text
POST /api/video-tasks/status
```

前端每 5 秒查询一次。

如果返回：

```text
pending / submitted / running
```

继续轮询。

如果返回：

```text
succeeded + video_url
```

调用 `completeGeneration()`，生成视频资产。

如果失败，调用 `markGenerationFailed()`。

## 12. 素材资产库 vs 丽帧资产库

### 12.1 素材资产库

作用：

- 漫镜内部素材预览/素材管理。
- 图片、视频、音频、SD2素材分类。
- 支持本地上传预览。
- 支持公网 URL 创建丽帧资产。
- 支持手动绑定已有 `asset://id`。

限制：

- 本地上传只支持页面预览。
- 不能直接用于真实视频生成。
- 因为丽帧资产创建接口只接受公网 URL。

### 12.2 丽帧资产库

作用：

- 从丽帧 API 查询已上传资产。
- 展示资产状态。
- 勾选用于 Seedance 生成。
- 当前用户主要使用这里的素材生成视频。

操作入口：

```text
详情
用于生成 / 取消使用
删除
```

注意：删除当前仍是占位 alert，未真正接入 `/asset/delete`。
